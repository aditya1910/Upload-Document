# Upload-Document

There are 2 folders associated with it 

I. client for the frontent 
to setup client 

1. cd client 
2. yarn install
3. yarn start


II. backEndApi fro the REST API
to setup backEndApi

1. cd backEndApi
2. npm install
3.npm start or nodemon

System requirements 

Node version 8.11.3 or greater 
MongoDb 3.4 or greater
